# SyncUp - Commit Test Point

Este archivo se creó para probar el flujo de `git pull` y `git push` en tu entorno local.

Pasos sugeridos:
1. Ejecuta `git pull origin main` y verifica que este archivo aparezca en tu copia local.
2. Edita este archivo (por ejemplo, agrega tu nombre y la fecha abajo) y guarda.
3. Haz commit y push:
   ```bash
   git add README-COMMIT-TEST.md
   git commit -m "test: verifying local git push works"
   git push origin main
   ```
4. Refresca el repo en GitHub para validar que se reflejó el cambio.
